# IronPDF C++ Beta Program Landing Page

## Project Overview

This project is a responsive landing page for the **IronPDF for C++ Beta Software Program**.  
The page was developed based on a provided **Figma design** and implemented using **CodeIgniter 4**, following the MVC architecture.

The goal of this project is to demonstrate the ability to:

- Convert a **Figma design into responsive HTML/CSS**
- Implement the page within a **CodeIgniter project**
- Load dynamic content using a **JSON file**
- Follow **clean code structure and web performance best practices**

The page contains several sections including:

- Hero section
- Product features
- Information section
- Why C++ PDF library section
- Early access section
- Beta program signup form

# Setup Instructions

## 1. Clone the Project

```bash
git clone <repository-url>
```
## 2. Install Dependencies

Run composer to install CodeIgniter dependencies:
```bash
composer install
```
## 3. Run the Development Server

Inside the project directory run:
```bash
php spark serve
```

The application will start at:
```bash
http://localhost:8080
```
